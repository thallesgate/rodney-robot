O Rodney Interface utiliza o ROS (Robot Operating System) que é uma coleção de frameworks de software para desenvolvimento de um sistema operacional para robôs.
Dentro desse sistema possuímos os chamados 'Nós' (ou Nodes), que são programas com uma função específica. Os 'Nós' conseguem trocar informações entre si através de tópicos e mensagens, esses tópicos são como se fossem o título da conversa, e as mensagens são as informações trocadas, podendo ser de todo tipo, como: textos, vetores e até imagens. A página da web utiliza bibliotecas que se comunicam com o sistema do robô para receber e enviar mensagens.
É assim que podemos controlá-lo em tempo real, ver imagens da câmera, receber um retorno dos sensores renderizando um mapa, além de enviar comandos e receber dados do robô.
